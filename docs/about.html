About This Project

This site adopts a React‑free, dependency‑minimal approach similar to the original Amos static conversion.

Every page is a real HTML file. Styling and behavior are kept intentionally small for speed and clarity.

Principles

  • No runtime framework overhead.
  • Readable, portable markup and CSS.
  • Progressive enhancement rather than heavy tooling.
  • Minimal bundle surface: one CSS file, one JS file.
